# Contributing to Pibi

> 🇨🇴 Spanish speakers: see [docs/es/contribuir.md](docs/es/contribuir.md)

Thanks for your interest. Pibi is a small hardware + firmware project — contributions of all kinds are welcome: code improvements, new character support, hardware variants, documentation.

## Language policy

| Element | Language |
|---------|----------|
| Code (variables, comments) | English preferred |
| Commits and PR titles | English |
| Issues | English or Spanish, both accepted |
| Documentation | English or Spanish, both welcome |

## How to contribute

1. Fork the repository
2. Make your changes
3. Open a Pull Request with a clear description of what changed and why

## Ideas for contributions

- Add support for accented characters (á, é, í, ó, ú, ñ)
- Add support for numbers (Braille numeric indicator)
- Adjustable timing (replace hardcoded 400ms delay)
- Multi-cell display support
- Companion desktop app for text input

## Reporting issues

Open an issue at https://github.com/Fedgutcor/Pibi/issues with:
- Your Arduino board and IDE version
- Steps to reproduce
- Expected vs actual behavior
