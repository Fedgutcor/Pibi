# Changelog

## [0.1.0] — 2018-09-09

### Initial release

- Arduino firmware for digital-to-Braille conversion
- Serial input (115200 baud) → 6-solenoid Braille cell output
- Support for all 26 letters (a–z) and space
- Automatic lowercase conversion
- 400ms activation time per character
- Developed as undergraduate thesis at ITM, Medellín, Colombia
