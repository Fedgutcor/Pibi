# Cómo contribuir a Pibi

Gracias por tu interés. Pibi es un proyecto pequeño de hardware + firmware — se aceptan contribuciones de todo tipo: mejoras al código, soporte de nuevos caracteres, variantes de hardware, documentación.

## Política de idiomas

| Elemento | Idioma |
|----------|--------|
| Código (variables, comentarios) | Inglés preferido |
| Commits y PR titles | Inglés |
| Issues | Inglés o español, ambos aceptados |
| Documentación | Inglés o español, ambos bienvenidos |

## Cómo contribuir

1. Haz fork del repositorio en GitHub
2. Realiza tus cambios
3. Abre un Pull Request con una descripción clara de qué cambió y por qué

## Ideas para contribuir

- Soporte de caracteres acentuados (á, é, í, ó, ú, ñ)
- Soporte de números (indicador numérico Braille)
- Tiempo de activación ajustable (reemplazar el delay fijo de 400ms)
- Soporte para display multi-celda
- App de escritorio complementaria para entrada de texto

## Reportar problemas

Abre un issue en https://github.com/Fedgutcor/Pibi/issues con:
- Tu placa Arduino y versión del IDE
- Pasos para reproducir el problema
- Comportamiento esperado vs. lo que pasó
